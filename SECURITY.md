# Security Policy

## Supported Versions

Security fixes are applied to the latest release line of `@kryv/teal`.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report them privately through GitHub's private vulnerability reporting:
<https://github.com/platypus27/teal/security/advisories/new>

We aim to acknowledge reports within 3 business days. For confirmed
vulnerabilities we target a fix release within 14 days, faster for
critical severity.

Reporters are credited in the release notes unless they prefer to remain
anonymous.
